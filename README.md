# pocketwatch: a Deno CLI time tracker

**Prerequistes**
- [Deno CLI](https://deno.land)
- That's it. Literally.

**Usage**
- Download the pocketwatch.ts file and place it _somewhere_
- Use your terminal in that directory
- Run `deno run --allow-all pocketwatch.ts` & Enjoy!
